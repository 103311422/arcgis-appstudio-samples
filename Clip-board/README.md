
## Clip-board
On Map tab:
You can click on the "+" button and choose any of the clipboards operations, and then paste the result in either Data or Image tab

On Copy tab:
- Text Field section,
You can type any text on the text area and click Copy button, then you can see the result in Data tab

- Image section
You can click on Copy image and paste the image in Image tab
You can also Copy the image URL and paste it in Data tab

- Color section
You can choose the color the rectangle
Click Copy button and the color would reflected on the next rectangle

On Copy tab:
This is where you see the pasted text

On Image tab:
This is where you see the pasted image

## Instructions to run this sample in AppStudio Desktop

1. Download the `.zip` file
2. Unzip and copy this folder into AppStudio Apps folder (Windows: `C:\Users\<username>\ArcGIS\AppStudio\Apps` Mac or linux: `Home\ArcGIS\AppStudio\Apps`)
3. The new app will now appear in the AppStudio Desktop. Run the application or open it in the bundled Qt-Creator IDE to look at the code and modify.
